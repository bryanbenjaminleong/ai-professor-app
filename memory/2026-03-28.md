## Technical Issues Encountered & Solutions

### Issue 1: Lesson page not loading
- **Cause:** `const router = useRouter()` declared but `useRouter` not imported
- **Fix:** Removed unused router declaration

### Issue 2: Admin page redirecting to /dashboard
- **Cause:** Multiple issues:
  1. Route group `(admin)` layout conflicts
  2. `/api/auth/session` endpoint didn't exist (using NextAuth)
  3. NextAuth configured for custom signin page that didn't exist
- **Solution:**
  1. Moved admin to `/analytics` URL
  2. Used `useSession()` from next-auth/react
  3. Created `/auth/signin/page.tsx` with Google login
  4. Use `callbackUrl=/analytics` in signin URL

### Issue 3: 404 on /api/auth/signin
- **Cause:** NextAuth configured with `pages.signIn: '/auth/signin'` but page didn't exist
- **Fix:** Created signin page at `/auth/signin/page.tsx`

## Database Changes

### Ratings Table (created in Supabase)
```sql
CREATE TABLE ratings (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id) NOT NULL,
  course_id UUID REFERENCES courses(id) NOT NULL,
  rating INTEGER NOT NULL CHECK (rating >= 1 AND rating <= 5),
  review TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(user_id, course_id)
);
CREATE INDEX idx_ratings_course ON ratings(course_id);
CREATE INDEX idx_ratings_user ON ratings(user_id);
```
